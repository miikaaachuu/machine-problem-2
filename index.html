ABOUT THE GAME

Venture forth into the Memory Dungeon, where demons lurk waiting for another ruthless killing. Remember as much as you can before all the cards turn around. Kill demons by matching them with their kin, and earn more lives by finding and matching potions.

Stay alive, find all the demons, and explore the depths of the memory dungeon.

HOW TO INSTALL

The whole game can be copied to your directory by downloading the source code or by cloning the git repository like so:


				git clone http://www.github.com/lickorice/cs11-mp2.git

Also, make sure install the requirements needed:


				pip install -r requirements.txt

To start the game, run the game through Python 3 in the command terminal with the following command (given that you are in the root directory of the program):

On Windows:


				py -3 main.py

On Linux:


				python3 main.py
