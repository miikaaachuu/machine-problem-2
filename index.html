ABOUT THE GAME

Create stories from totally random words that you input! Best played with friends.

HOW TO INSTALL

Go to the website below and download all the files and resources:


				www.github.com/miikaaachuu/mad-libs

Open your command prompt and go to the directory where you downloaded the files like so:


				If you downloaded it in the Documents folder:
				
				cd Documents
				
				If you downloaded it elsewhere:
				
				cd maindirectory\folder

To start the game, run main.py.


				main.py
